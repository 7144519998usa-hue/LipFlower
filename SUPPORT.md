# Support

LipFlower is a publisher-grade beauty affiliate and education site. Use GitHub issues for project work, not consumer beauty advice.

## Use GitHub Issues For

- Route bugs
- Build or deployment problems
- SEO registry or sitemap issues
- Programmatic page quality problems
- Affiliate disclosure or outbound redirect concerns
- Beauty claims, structured data, or compliance fixes

## Do Not Use GitHub Issues For

- Personal skin, hair, health, or medical advice
- Product safety decisions for an individual shopper
- Private affiliate account credentials
- Sensitive security reports with exploit details

## Before Requesting Support

Run:

```bash
npm run verify
```

If the issue affects production, also run after deployment:

```bash
npm run verify:production
```
